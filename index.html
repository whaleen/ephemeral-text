Ephemeral Text

Write freely. Nothing is saved unless you export it. Grammar hints are inline and optional.

Features

  • Rich Markdown editing with live formatting
  • Inline grammar hints with click-to-apply fixes
  • Manual export to .md or .txt (no autosave)
  • Custom export directory selection
  • Light/dark theme toggle
