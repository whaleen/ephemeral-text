# ephemeral-text

textEdit alternative for Mac OS that doesn't save files, and "doesn't ask any questions".

it will export a txt or md file for you if you want and it will not overwrite it. it will export addiotnal files with a number suffix. it will not ask you to save when you close the window. it will not allow you to hve more than one windo open. it will not ask you to save when you quit the app. it will not ask you to save when you open a new window.
